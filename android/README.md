### org.apache.httpcomponents 引用问题


### 解决安装后出现多个图标问题
注释掉以下library模块的AndroidManifest.xml以下内容

```
<!--<intent-filter>-->
               <!--<action android:name="android.intent.action.MAIN"/>-->
                <!--<category android:name="android.intent.category.LAUNCHER"/>-->
<!--</intent-filter>-->
```
